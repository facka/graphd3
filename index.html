Help:

Click anywhere: add a new Node
Ctrl + drag a node: creates a link to:
    - existing node if the link is dropped over a node
    - new node if the link is not dropped over a node
Drag a node: moves it
Key R:Set right direction to the selected link
Key L:Set left direction to the selected link
Key B:Set both directions to the selected link
Key supr/delete:Deletes the selected item. If the selected item is a node also deletes all its links
